# rn-tabmotion

A customizable and lightweight bottom tab bar component for React Native applications, designed for smooth animations and ease of use.

## Installation

```bash
npm install rn-tabmotion
```

## Usage

```javascript
import { TabMotion } from 'rn-tabmotion';

const App = () => {
  return <TabMotion />;
};

export default App;
```
